介绍

swagger 是一套开源的API设计工具，包括Swagger UI，Swagger Editor等。

其中Swagger Editor是个WEB小程序，它可以让你用YAML来定义你的接口规范，并实时验证和现实成接口文档。

我们主要使用Swagger Editor 编写接口文档，在线 Swagger Editor 也是可以使用的，但是访问速度在国内有点慢，所以为了效率，自己在服务器搭建一套服务
