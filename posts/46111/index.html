Spring Boot 与 Kotlin 验证web表单信息

Spring Boot与kotlin实战 kotlin 2018/02/03

在做web开发的时候，我们需要验证表单，确认用户提交的信息是安全的，比如用户名不能超过多少位，密码不能少于多少位等等。

那么如何在Spring Boot 与 Kotlin中验证表单信息？

在springmvc工程中，需要检查表单信息，表单信息验证主要通过注解的形式。
